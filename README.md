# Skills

A simple toolbox of skills for solving real-world problems with Claude.

Each skill is a self-contained folder you can drop into your Claude.ai project. No complex setup — just upload and use.

## Skills

| Skill | What it does |
|-------|-------------|
| [skill-inspector](./skill-inspector) | Inspect and iteratively improve an existing skill through structured goal-setting |
| [export-project](./export-project) | Export all project knowledge into a package optimized for Claude.ai, Claude Code, ChatGPT, or Cursor |
| [project-bootstrap](./project-bootstrap) | Bootstrap a project's knowledge layer through structured interview, generating cache-friendly docs for LLM and human consumption |

## Usage

1. Download a skill folder
2. In your Claude.ai project, upload the `SKILL.md` (and any `scripts/` files) to Project Files
3. Use it in conversation — Claude picks it up automatically from trigger phrases
